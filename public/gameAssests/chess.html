Set time, then press Start White/Green. Players tap their own side to switch.
How to use:
  • Set minutes and seconds, then click Set Time.
  • Start the game with Start White or Start Green.
  • When your move is done, tap your own side button: your clock stops, opponent’s starts.
  • Pause All stops both clocks; Reset sets both back to initial time.
